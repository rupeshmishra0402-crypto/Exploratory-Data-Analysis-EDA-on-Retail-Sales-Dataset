# Exploratory-Data-Analysis-EDA-on-Retail-Sales-Dataset
Worked on a retail sales dataset to understand product and region-wise performance. Cleaned and structured the data, then explored patterns in sales and profit. Created simple visualizations to highlight trends. This project helped me turn raw data into meaningful insights for better decision-making.
# Sales Performance Analysis & Business Insights

## Objective
To analyze sales data and identify key trends.

## Tools Used
- Python (Pandas, Matplotlib, Seaborn)

## Key Insights
- North region has highest sales
- Laptop is best-selling product

## Conclusion
This project helps in making data-driven business decisions.
